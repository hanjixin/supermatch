# 弹出层组件

LayerPop

## 版本

v1.0.0

## 参数

- width
  - number
  - 宽度 小于 100 默认为百分比，大于 100 为 px
  - DefaultValue 80
- padding
  - string
  - paddingTop And padding Bottom
  - DefaultValue '0'
- visible
  - Boolean
  - 是否显示弹出层 true 为显示
  - DefaultValue false
- children
  - ReactNode
  - 需要弹出的内容
  - DefaultValue () => <></>
